Mac 解决方案

  1. 找到你电脑的这个文件夹 /private/etc（点击finder, 顶部菜单栏：前往-前往文件夹，输入 /private/etc/hosts）
  2. 将里面的 hosts 文件复制一份（因为不能直接修改，然后修改了之后再覆盖原来的 hosts）
  3. 跟 windows 一样，在 hosts 里加入 ip 映射地址
  4. 刷新dns：sudo killall -HUP mDNSResponde
